# An GRPC Server and Clint running on Flutter
## Instructions
First install dart and flutter (google it)
```git clone https://github.com/eudangeld/flutter-grpc.git```


## 1) GRPC pack 
```cd grpc ```
```pub get ```

## 2) Server  
```cd server ```
```pub get ```
```dart bin/server.dart ```

## 3) Flutter 
```cd flutter ```
```flutter pacakges get ```
```flutter run ```
